# IoT Cyber-Hygiene Assistant (Frontend)

A Project-Based Learning (PBL-II) (IVth SEM) frontend dashboard for scanning IoT devices (demo simulation), assigning a Cyber-Hygiene Risk Score (0–100), generating insights, and exporting reports.

## Features
- Simulated IoT device scan
- Risk scoring system (0–100)
- Vulnerability analysis & recommendations
- Device inventory table with search/filter/sort
- Charts for risk distribution
- Export report (JSON / CSV)
- GitHub Pages hosting ready

https://kavins-glitch.github.io/iot-cyber-hygiene-ui-pbl-project/
